DOM이란?

DOM(Document Object Model : 문서 객체 모델)은 HTML 문서를 읽고 조작하는 객체의 집합임. 즉 HTML 문서의 구조를 말하며, 자바스크립트를 사용하여 문서객체를 선택하고, 속성 또는 스타일을 적용함

JQuery란?

HTML DOM(Document Object Model)과 자바스크립트 사이의 상호작용을 간단하게 해주는 오픈소스 자바스크립트 라이브러리. 'javascript(j)에서 DOM 엘리먼트를 질의(Query)한다.' 이 문자에서 j와 Query를 합친 단어 jQuery 함수는 jQuery(); 또는 $();를 말하며, DOM에서 엘리먼트(요소, 태그)를 선택하기 위해서 사용한다. jQuery 래퍼(wrapper)라고도 불리며, 메소드의 매개변수로 CSS 셀렉터 표현식을 문자열 형태로 기술한다. $(document).ready(); => html 문서가 웹브라우저에 로드되어 준비된 상태가 되었을 때, 적당한 처리를 하기위한 이벤트 핸들러이다. "window.onload"를 대신하여 jQuery가 제공하는 메소드로서, DOM이 로드되고 이미지가 로드되기 전에, 호출되는 이벤트 핸들러로 준비핸들러라고 함 http://jquery.com/download에서 Download the compressed, production jQuery 3.4.1를 우클릭해서 다른 이름으로 링크저장 => js폴더 생성 후 jQuery 3.4.1.min.js 다운로드 저장
